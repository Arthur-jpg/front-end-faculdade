Git - básico
    - Se eu for modificar alguma coisa no arquivo eu bloqueio para ninguem modificar tb
    - Controle do estado de arquivo
        - Toda vez que você for modificar o arquivo você baixa a versão mais nova dele para n ter problema de versão
        - MD5 -> algorítimos para gerar hash (um número que representa algo)
            - Você passa um dado para o mad5 e ele te dá um número de 128 bits
            - Se você quiser o dado novamente você só consegue se tiver a sequência de 128 bits novamente
        - O git olha para o arquivo de 3 formas
            - Versionado()commited() -> quando está up to date
            - modificado(modified) -> quando está vermelho
            - preparado(staged) -> quando ta verdinho 
        - Fluxo base de trabalho
            - Modificar
            - Preparar
            - Fazer commit
